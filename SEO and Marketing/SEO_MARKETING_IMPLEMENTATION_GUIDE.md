# RENDER DEPLOYMENT OPTIMIZATION

Since the website is hosted on Render Free Tier:

- Enable Brotli/Gzip compression where possible.
- Configure aggressive caching for static assets.
- Ensure proper cache-control headers.
- Prevent unnecessary cold-start requests where possible.
- Optimize bundle size using code splitting.
- Serve images in modern formats (WebP/AVIF).
- Add prerendering or SSR if supported by the architecture.
